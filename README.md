# ChemBio with Yoo — Unit Quiz Site

## 로컬에서 테스트 (선택사항, 코딩 몰라도 넘어가도 됨)
```
npm install
npm run dev
```

## 배포 방법 (GitHub + Vercel)
1. 이 폴더 전체를 GitHub 저장소에 업로드
2. https://vercel.com 가입 → GitHub 연동 → 이 저장소 Import
3. Vercel이 자동으로 Vite 프로젝트를 감지해서 빌드/배포함 (설정 변경 불필요)
4. 배포 완료 후 Vercel이 주는 기본 주소(xxx.vercel.app)로 확인 가능
5. Vercel 프로젝트 → Settings → Domains → 원하는 도메인 입력 → 안내되는 DNS 값을 도메인 구매처(가비아 등)에 등록
